01 · Python · Pandas · Matplotlib · Jupyter

Sales Dashboard Analysis

Cleaned and analysed 9,994 retail sales records spanning four years (2014–2017) from the Superstore dataset, then built a 6-chart interactive dashboard to surface revenue growth and regional performance trends.

Correlation analysis (r = −0.22 between discount and profit) identified the Tables sub-category as a $17,725 loss driver — discounted at an average of 30% versus 15% across the rest of the catalogue — and turned that into an actionable pricing recommendation.

02 · SQL · PostgreSQL

Social Media Marketing Analytics

Designed a 6-table normalised PostgreSQL schema (users, campaigns, posts, engagements, ad spend, conversions) and wrote 10 advanced SQL queries — CTEs, window functions, and cohort logic — to analyse campaign ROI, engagement funnels, and content performance.

Built a composite efficiency-scoring model using min-max normalisation across CTR, ROAS, and engagement rate, plus NTILE()-based customer LTV segmentation (Champions / Loyal / At-Risk), mirroring a real Marketing Data Analyst workflow.

03 · Python · Scikit-learn · Pandas

Smart Fashion Recommender

Built a collaborative-filtering recommendation engine on a synthetic dataset of 10,000+ users and 500 clothing items, covering EDA, feature engineering, and model evaluation.

Compared four models — KNN-based collaborative filtering and SVD at different component counts — via hyperparameter tuning, with the best configuration (CF-KNN, k=20) reaching Precision@5: 0.87 and an F1-score of 0.75.

04 · Python · Tableau · MSc Dissertation

Visualising Logical Cyber Security Arguments

My MSc dissertation at the University of Strathclyde: an interactive tool that visualises Attack-Defence Trees and Dung-style argumentation semantics for ePassport (ICAO 9303) security, making formal security arguments understandable to non-technical stakeholders.

Modelled 6 attack vectors and 5 defences with DREAD scoring, and evaluated the interactive prototype against static reference material using the System Usability Scale and comprehension testing — the interactive version scored notably higher on both usability and comprehension.
